# 📊 Comparação entre Salário Mínimo e IPCA (2010–2020)

Projeto desenvolvido em JavaScript com Node.js para analisar a evolução do salário mínimo brasileiro e compará-la com os índices de inflação (IPCA) entre 2010 e 2020.

## 🚀 Funcionalidades

* Listar os salários mínimos de 2010 a 2020;
* Listar os índices anuais do IPCA;
* Comparar o percentual de aumento do salário mínimo com o IPCA de cada ano;
* Interface interativa via terminal.

## 🛠️ Tecnologias

* JavaScript
* Node.js
* Readline (módulo nativo)

## 📂 Estrutura dos Dados

O projeto utiliza arrays de objetos para armazenar:

* Valores do salário mínimo por ano;
* Índices IPCA por ano.

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd nome-do-projeto
```

3. Execute o arquivo:

```bash
node index.js
```
